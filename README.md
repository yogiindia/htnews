# Hacker News App build using VueJS

This app shows upto 500 top stories.

[Demo](http://htnews.surge.sh/)

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
